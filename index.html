Box Model Playground

Each colored box shows how content, padding, border, and margin affect space. All boxes share the same content width, but different padding/border values change the total size.

Content only

No padding, 0 border, small margin. Tight like morning BRT schedules.

Padded

Added padding creates breathing room around text — like shade under a palm tree.

Padded + border

Border visually defines the edge; margin separates this box from neighbors.

Tip: With box-sizing: border-box;, declared widths include padding and border for predictable sizing.
